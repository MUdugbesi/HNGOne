My Two years goals

React.js

2024-2025
  • Master advanced React concepts such as Context API, React Hooks (useEffect, useContext, useReducer), and Custom Hooks.
  • Build projects that utilize these concepts to solve real-world problems.
  • Learn, implement and Refactor existing projects to use Redux/MobX for better state management.
  • Learn and implement testing in React using tools like Jest and React Testing Library.
  • Learn performance optimization techniques such as code splitting, lazy loading, and memoization.
  • Explore TypeScript with React and Optimize Performance
  • Attend Workshops and Conferences
2025-2026
  • Learn back-end technologies (Node.js, Express, MongoDB) to create full-stack applications. Deploy applications using services like AWS, Heroku, or Vercel.
  • Contribute to React or other open-source projects on GitHub. Engage with the developer community and collaborate on larger projects.
  • Learn advanced CSS techniques, CSS-in-JS libraries like styled-components or Emotion.
  • Attend React conferences, webinars, and workshops to stay updated with the latest trends and best practices.
  • Regularly follow React documentation, blogs, and community forums.
  • Obtain Certifications

Tailwind Css

2024-2025
  • Master the utility classes in Tailwind CSS and learn how to customize the Tailwind configuration to match specific design requirements.
  • Create complex grid layouts and responsive designs using Tailwind CSS. Implement common UI patterns (e.g., navigation bars, cards, modals) using Tailwind.
  • Contribute to Tailwind CSS-related open-source projects. Share custom components and configurations with the community.
  • Study and implement advanced CSS techniques such as CSS Grid and Flexbox with Tailwind.
2025-2026
  • Optimize Tailwind CSS for performance, including purging unused styles and optimizing build times. Learn to measure and improve the performance of Tailwind-based projects.
  • Create a comprehensive design system using Tailwind CSS. Ensure consistency across different projects by adhering to the design system.
  • Dive deeper into advanced customization of Tailwind CSS, including creating custom plugins. Explore dark mode and theme switching capabilities.
  • Share knowledge through blog posts, tutorials, or workshops. Mentor junior developers in using Tailwind CSS effectively.
